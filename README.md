# ONJE

Student kitchen order dashboard for local use.

## Run

- `npm install`
- `npm run dev` for the live app
- `npm run build` to compile a local preview build
- `npm run preview` to serve the compiled build

## Notes

- `npm run dev` binds to `127.0.0.1:4180`
- `npm run preview` binds to `127.0.0.1:4181`
- Build output is in `build/`
